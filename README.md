# Task Tracking System

A comprehensive task management system with dual portals for team members and administrators.

## Features

- **Dual Portal System**: Separate interfaces for team members and administrators
- **Zone-based Task Management**: 30+ predefined zones for task assignment
- **Role-based Access Control**: Three user roles (Team, Admin, System Admin)
- **Task Approval Workflow**: Multi-level approval system with audit trail
- **Excel Export**: Export task details to Excel format
- **Real-time Updates**: Live task status updates
- **Priority Management**: Mark tasks as urgent/normal/low priority
- **Full Visibility**: All tasks visible to authorized users
- **System Administration**: Full control for system admins

## User Roles

### 1. Team Members
- Request new tasks
- View all tasks
- Update task status
- Mark tasks as complete

### 2. Administrators
- All team member features
- Approve/reject tasks
- Create and assign tasks
- Mark tasks as urgent
- Export data to Excel
- View system statistics

### 3. System Administrators
- All administrator features
- Edit/delete any task
- Manage users
- View system logs
- Full system control
